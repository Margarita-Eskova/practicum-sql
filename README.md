## 📊 Краткое описание работ

### Лабораторная работа №1
Освоение базовых операторов SELECT, фильтрации, сортировки и CRUD-операций в локальной среде.

### Лабораторная работа №2
Изучение объединения таблиц (JOIN, UNION), подзапросов и функций преобразования данных (CASE, COALESCE).

### Лабораторная работа №3
Освоение агрегации данных (COUNT, SUM, AVG, GROUP BY, HAVING) для анализа качества данных.

### Лабораторная работа №4
Изучение оконных функций (RANK, LEAD, AVG с оконным фреймом) для аналитических расчётов.

### Лабораторная работа №5
Анализ производительности запросов с помощью EXPLAIN ANALYZE и оптимизация с использованием индексов B-Tree.

### Практическая работа №1
Применение продвинутых возможностей PostgreSQL: работа с датами, JSON, массивами и текстовой аналитикой.
